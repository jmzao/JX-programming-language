Comandos Básicos

• Console.display("texto") - Mostra no console
• Console.input() - Recebe entrada
• Wait(time) - Espera tempo em ms
• Loop {} - Estrutura de loop
• stop() - Para o loop

Renderização

• Console.back.backspace - Apaga caractere
• Console.back.enter - Nova linha
• back.line(n) - Volta linha n (descarta estados)
• back.linesave(n) - Volta linha n (salva estados)

Operadores

• == (igual), #= (diferente)
• <, >, +, -, *, /
• "texto" + "texto" (concatenação)
